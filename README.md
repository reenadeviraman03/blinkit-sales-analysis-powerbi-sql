# 📊 Blinkit Sales Performance Analysis

## 📌 Project Overview
This project analyzes Blinkit sales data using PostgreSQL and Power BI.
The objective is to identify sales trends, revenue performance, and key business insights.

---

## 🛠 Tools Used
- PostgreSQL
- Power BI
- SQL
- Excel (for dataset preparation)

---

## 📊 Key KPIs
- Total Sales
- Average Sales
- Number of Orders
- Average Rating
- Sales by Item Type
- Sales by Outlet Type
- Sales by Fat Content

---

## 📈 Dashboard Features
- Interactive Filters
- Category-wise Sales Analysis
- Outlet Performance Comparison
- Year-wise Sales Trend
- Sales Percentage Distribution

---

## 🧹 Data Cleaning
- Standardized Item Fat Content
- Handled inconsistent values
- Removed duplicates (if any)

---



## 📌 Business Insights
- Identified top-performing outlet types
- Analyzed customer rating impact on sales
- Compared fat content sales performance
- Observed yearly sales growth patterns

---

## 🚀 Conclusion
This project demonstrates SQL querying, data transformation, and dashboard development skills suitable for Data Analyst roles.
